Функции:
- запись информации об услугах
- Корректировка услуг
- Удаление услуг
- Вывод инфорации хранящейся в базе

Доп функции:

- Принимает данные из файла json расположенного в папке с программой
- Автоматически вносит изменения в базу при каждой успешной операции.


Создание записи

- Автоматическое с начальными данными услуга = 0
- Проверка на уже существующую запись из принятой базы



Удаление записи

- Возможно удаление всей записи за период
- Возможно обнуление значения за услугу
- Автоматическое сохранение
